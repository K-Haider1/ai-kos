# Setup Guide

## Prerequisites

* Python 3.11
* Git
* VS Code

## Clone Repository

```bash
git clone <https://github.com/K-Haider1/ai-kos.git>
cd ai-kos
```

## Create Virtual Environment

```bash
py -3.11 -m venv .venv
```

## Activate Virtual Environment

### Windows PowerShell

```powershell
.\.venv\Scripts\Activate.ps1
```

## Install Dependencies

```bash
pip install -r requirements.txt
```
